# Basic Streamlit Chatbot

A minimal chatbot template with simulated responses.

## Quick Start

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the app:
```bash
streamlit run app.py
```

3. Open your browser to `http://localhost:8501`

## Features

- Clean chat interface with message history
- Simulated streaming responses with typewriter effect
- Clear conversation button
- Debug info panel

## Customization

Replace the `simulate_response()` function with your LLM API calls to create a real chatbot.
